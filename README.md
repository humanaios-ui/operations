## Zone system (plain language)

- **Zone 1 — AI executes:** the AI can perform scoped repository work under protocol constraints.
- **Zone 2 — operator decides:** the operator makes authorization and policy-boundary decisions.
- **Zone 3 — operator executes in terminal:** credential-bearing, external, or safety-sensitive terminal actions are executed by the operator.
