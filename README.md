# HumanAIOS Operations Repository

Canonical governance and research infrastructure repository for HumanAIOS.

## Repository role

This repository is the canonical source for governance state, execution protocol, and research infrastructure documents used across HumanAIOS surfaces.

## Canonical documents and authority

| File | Authority role |
|---|---|
| `CURRENT.md` | Session-open operating state and architecture pointers |
| `GOVERNANCE.md` | Standing principles and authority hierarchy |
| `SESSION_RITUALS.md` | Required cross-substrate open/close protocol |
| `REGISTERED.md` | Append-only findings, hypotheses, and incident registry |
| `OPERATOR_RUNBOOK.md` | Public operator procedures and execution recipes |

Additional protocol surfaces: `Z3_PROTOCOL.md`, `SUBSTRATE_CAPABILITY_REGISTRY.md`, `ACAT_SESSION_PROMPT.md`.

## Zone system (plain language)

- **Zone 1 — AI executes:** the AI can perform scoped repository work under protocol constraints.
- **Zone 2 — operator decides:** the operator makes authorization and policy-boundary decisions.
- **Zone 3 — operator executes in terminal:** credential-bearing, external, or safety-sensitive terminal actions are executed by the operator.
